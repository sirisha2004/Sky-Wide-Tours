# ✈️ Skywide Tours

A modern and responsive travel website built using **HTML**, **CSS**, and **JavaScript**. It features destination packages, services, gallery, reviews, and a contact form — all wrapped in a clean, user‑friendly UI.

## ✨ Features
- 🎞️ Hero video section  
- 📱 Responsive navigation + mobile menu  
- 🔍 Search bar & login popup  
- 🧳 Tour packages with pricing and ratings  
- 🖼️ Image gallery with hover effects  
- ⭐ Customer review slider  
- ✉️ Contact form UI  

## 📁 Project Structure
- skywide travel.html   → Main webpage
- style.css             → Styling & responsive layout
- script.js             → Menu, search, login & slider logic
- assets/               → All images and videos

## 🛠️ Tech Stack
- HTML
